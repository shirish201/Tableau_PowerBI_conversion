Phase 1 - Data Model: Import Excel data, set up relationships, create 10 parameter tables, build a Date dimension, and enter all DAX measures in dependency order.

Phase 2 - Descriptive Page: The main KPI dashboard with 5 region maps, 3 KPI columns (Sales, Profit Ratio, Days to Ship) each containing a Card callout, Bullet Chart (AppSource), Line Chart  
trend, and Dumbbell/Bar region comparison.

Phase 3 - Prescriptive Page: Filled Map on the left with Map KPI Difference coloring, Scatter Chart on the right with dynamic X/Y axes via parameter slicers, plus Field Parameters for the    
dimension switcher.

Phase 4 - Annotations Page: Insight indicator circles with conditional formatting (green/amber/red) and text cards, with notes on how to handle free-text input limitations in Power BI.       

Phase 5 - Interactivity: Cross-filtering, page navigation buttons, slicer sync across pages, and custom tooltip pages.

Phase 6 - Styling: Full color theme (#34657f teal, #fc4237 red, #00313c text), Corbel/Calibri fonts, gridline settings, and the diverging color palette.

Phase 7 - Testing: Data validation against the original Tableau workbook, interactivity testing, and formatting verification.
